NeoPixels

A.K.A. WS2812, WS2811 or SK6812

“NeoPixel” is Adafruit's brand for individually-addressable RGB color pixels and strips based on the WS2812, WS2811 and SK6812 LED/drivers, using a single-wire control protocol.

What is it

  • Driving lots of RGB leds used to be "hard"
  • .. not anymore with the WS2812 LED (NeoPixel)
  • RGB LED with integrated IC
  • Single wire protocol
  • Individually addressable
  • Cheap (~30$ for strip of 300)

hard => required lots of external components, making it hard to make big strips or displays.

What is it?

How to drive it?

Feed it a bit signal with RGB values for each LED in the strip.

And 5V from a power source (for longer strips).

How to drive it?

With a (micro)controller:

  • Arduino
  • ESP8266/ESP32
  • Raspberry Pi
  • lots of other MCU's

Arduino ESP8266 (explain why this chip is awesome) ==> fast (160mhz) ==> WiFi ==> cheap (~2$ per chip or 5$ look for NodeMCU). ==> Arduino supported (can use whole ecosystem!) ==> MicroPython ==> Javascript (MongooseOS or Espruino) ESP32 ==> successor to the ESP8266 ==> dual core ==> has bluetooth ==> faster? ==> more flash?
